# SageBrush Wealth: Digital Assets Managed Like Wealth

## Getting Started

To get a local copy of the project up and running, follow these steps.

### Prerequisites
Ensure you have the following installed on your machine:
-   Node.js (LTS version recommended)
-   npm or Yarn

### Installation

Navigate to your desired directory and follow the commands below:

1.  **Clone the Repository**:
    ```bash
    git clone https://github.com/capitalsagetechnology/sbrush.git
    ```

2.  **Navigate to Project Directory**:
    ```bash
    cd sagebrush
    ```

3.  **Install Dependencies**:
    ```bash
    npm install
    # or
    yarn install
    ```

### Environment Variables
This frontend application primarily consumes external services and does not require specific environment variables for basic local development setup.

## Usage

Once the dependencies are installed, you can start the development server:

1.  **Start the Development Server**:
    ```bash
    npm run dev
    # or
    yarn dev
    ```

2.  **Access the Application**:
    Open your web browser and navigate to [http://localhost:3000](http://localhost:3000).

The application will reload automatically as you make changes. Explore the different sections like "ErcasPay," "ErcasCollect," "ErcasEvents," and "SageCloud" to understand the full suite of offerings provided to businesses.
